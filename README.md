# LITESCRIPT
Tools Untuk Membuat Script Deface Dengan Praktis

## Installation
```
$ apt update && upgrade
$ pkg install git
$ pkg install python2
$ pip2 install requests
$ git clone https://github.com/mrbaiz/lite-script
$ cd lite-script
$ python2 litescript.py
```
